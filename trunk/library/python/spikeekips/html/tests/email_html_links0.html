규칙은 첨부한 문서 보면 되는데, 간단히 요약하면.

  • 블럭(공)을 터치 드래그로 한 칸 이동시켰을 때 같은 색깔 블럭이 인접(x, y로 1칸씩)해 있으면 이동 됨
  • 같은 색깔 공이 3개 이상 이어지면 체인 걸린 상태. 이 체인 걸린 상태 공을 클릭하면 점수 획득. 체인이 길면 길수록 부가 점수가 큼.
  • 빨강, 노랑, 파랑색 공은 각각 클릭하여 인접한 서로의 색으로 섞어서 다른 색으로 만들 수 있음
나머지(?) 규칙들은 기본 규칙의 큰 틀에서 벗어나지 않으며, 예외 상황 대비 및 양념 같은 역할입니다.

게임 재미 목표는 

  1. 최대한 공을 길게 연결한 뒤 직접 터뜨리자! (그냥 터지면 기본 점수, 직접 터뜨리면 추가 점수 발생)
  2. 빨강, 노랑, 파랑을 섞는 걸 활용하자

이것이죠.

현재는 공(블럭) 생성 등 레벨 디자인이 거의 적용 안 되어 있어서 밸런스는 별로인데, 빨강, 노랑, 파랑공과
회색공을 활용해서 게임 박진감을 높일 수 있을 듯 합니다.
